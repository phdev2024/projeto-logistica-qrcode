# 📦 Sistema de Gestão de Etiquetas e Expedição

Este projeto automatiza a geração de etiquetas logísticas em PDF (100x50mm) e controla o fluxo de saída de materiais via QR Code.

## 🚀 Funcionalidades
- **Geração de Lotes:** Criação de etiquetas sequenciais em PDF.
- **Banco de Dados:** Armazenamento via SQLite para rastreabilidade.
- **Módulo de Expedição:** Baixa no estoque via leitura de QR Code.
- **Interface Web:** Desenvolvido com Streamlit.

## 🛠️ Tecnologias
- **Python 3.10+**
- **Streamlit** (Interface)
- **SQLite** (Banco de dados)
- **FPDF2** (PDFs)
- **Pillow & QRCode** (Imagens)
